# 微信自动发送群消息工具

这是一个基于 Python + PyAutoGUI + OpenCV 的 RPA（机器人流程自动化）工具，用于在本地电脑上自动操作微信 PC 版，向指定群聊发送消息。

> ⚠️ 注意：该工具仅适用于已登录的微信 PC 版，模拟用户操作，不依赖微信 API。

---

## 📦 功能简介

- **自动微信管理**：检测微信是否已启动，未启动则自动启动并登录
- **智能窗口操作**：自动切换到微信窗口，支持处理最小化窗口，焦点锁定确保操作不被打断
- **多种群聊定位**：支持多种查找群聊的方式（快捷键搜索、图像识别、坐标定位）
- **可配置优先级**：可以设置不同搜索方法的优先级，灵活应对不同环境
- **自定义快捷键**：支持自定义微信搜索快捷键，兼容不同版本微信
- **消息管理**：支持直接输入消息或自动发送消息
- **配置灵活**：通过配置文件修改群名、消息内容和行为模式
- **外部文件支持**：支持从外部文件读取消息内容，便于管理长文本
- **变量替换**：支持动态变量（如日期、时间）在消息中的自动替换
- **界面校准**：提供搜索框坐标校准功能，适应不同屏幕分辨率
- **详细日志**：提供全面的日志记录，方便排查问题

## 🧰 使用步骤

### 1. 克隆或创建项目目录

### 2. 准备虚拟环境

项目要求Python 3.7-3.11版本。请确保使用兼容版本创建虚拟环境：

```bash
# 使用特定Python版本创建虚拟环境（推荐）
python3.9 -m venv venv  # 或者其他3.7-3.11之间的版本

# 或使用系统默认Python（确保版本在3.7-3.11之间）
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate   # Windows
source venv/bin/activate # Linux/macOS
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 准备模板图片（已经提供好，不用准备）

如果您计划使用图像识别功能（搜索方式优先级2），需要准备以下模板图片：

1. **群聊搜索结果**：在微信中搜索目标群聊，截取左侧搜索结果的图片
   - 保存为：`png/group_result_template.png`
   - 如有必要，可准备备用模板：`png/group_result_backup.png`

2. **搜索图标**：截取微信主界面中的搜索图标
   - 保存为：`png/search_icon_template.png`
   - 如有必要，可准备备用模板：`png/search_icon_backup.png`

> 💡 提示：截图时尽量只包含目标元素及其少量周围内容，保持图片清晰，避免过多无关元素。

### 5. 修改配置文件 config.yaml

```yaml
# 微信安装路径（注意使用双反斜杠或原始字符串格式）
wechat_path: "D:\\app\\WeChat\\WeChat.exe"

# 目标群聊名称（需要完全匹配）
group_name: "AI研发工作室飙能群"

# 要发送的消息内容（当message_file不可用时使用）
message: "你好，这是一条自动发送的消息。"

# 可选：从外部文件读取消息（优先级高于message）
message_file: "message.txt"

# 是否自动发送消息（true-发送，false-仅输入不发送）
auto_send: true

# 校准等待时间（秒）
calibration_timeout: 10

# 搜索快捷键，格式为[modifier, key]，例如["ctrl", "f"]
search_shortcut: ["ctrl", "f"]

# 群聊搜索方式优先级: 1-快捷键搜索, 2-图像识别, 3-坐标定位
search_method_priority: [1, 2, 3]

# 搜索框坐标（由校准功能自动生成，通常不需手动修改）
# search_box_x: 745
# search_box_y: 273
```

#### 群聊搜索方式说明

工具支持三种不同的群聊搜索方式，可以通过设置优先级灵活应对不同环境:

1. **快捷键搜索(1)**：使用快捷键打开微信搜索，然后输入群名搜索。这是最可靠的方法，推荐优先使用。
   - 可通过`search_shortcut`配置项自定义搜索快捷键，默认为`["ctrl", "f"]`
   - 如果您的微信版本使用不同快捷键，请相应修改此配置

2. **图像识别(2)**：使用OpenCV图像识别技术定位群聊，需要预先准备模板图片。灵活性高但依赖于界面视觉一致性。

3. **坐标定位(3)**：使用预先校准的坐标位置操作界面，最简单但也最易受界面变化影响。

可以调整`search_method_priority`数组中的顺序来设置优先级。例如，`[1, 2, 3]`表示先尝试快捷键搜索，失败后再尝试图像识别，最后尝试坐标定位。

### 6. 准备消息文件（可选）

如果需要发送较长的消息或使用变量替换功能，可以创建`message.txt`文件：

```text
大家好！

这是一条从外部文件读取的测试消息。
当前时间: {time}
今天日期: {date}
周几: {weekday}

--自动发送
```

支持的变量：

- `{time}` - 当前时间（如：14:30:45）
- `{date}` - 当前日期（如：2025-05-16）
- `{datetime}` - 完整日期时间（如：2025-05-16 14:30:45）
- `{weekday}` - 星期几（如：Monday）

### 7. 校准搜索框坐标（可选）

如果您计划使用坐标定位搜索方式，需要校准搜索框坐标：

```bash
python main.py --calibrate
```

按照提示，在倒计时内（默认10秒，可在config.yaml中通过calibration_timeout参数修改）将鼠标移动到微信搜索框的中心位置。

### 8. 启动程序

标准模式：

```bash
python main.py
```

调试模式（显示更详细的日志）：

```bash
python main.py --debug
```

## ⚠️ 常见问题与解决方案

1. **无法正确定位群聊**
   - 尝试调整搜索方式优先级，优先使用快捷键方式 `search_method_priority: [1, 2, 3]`
   - 如果快捷键搜索失败，确认您的微信版本使用的搜索快捷键，并更新`search_shortcut`配置
   - 确保群名称在配置文件中拼写完全正确（包括空格等）
   - 如使用图像识别，更新模板图片确保清晰完整
   - 检查日志文件确认实际搜索的群名

2. **窗口焦点不稳定**
   - 确保没有其他程序抢占焦点
   - 减少后台运行的程序数量
   - 调整DELAY_SHORT和DELAY_LONG变量增加操作间隔

3. **权限问题**
   - 以管理员身份运行命令提示符或PowerShell
   - 确保程序有权限操作微信窗口和使用剪贴板

4. **消息输入问题**
   - 确保消息内容不包含特殊控制字符
   - 如果消息很长，优先使用message.txt文件

5. **提高运行成功率的技巧**
   - 保持桌面整洁，避免其他窗口干扰识别
   - 使用固定的微信窗口大小和位置
   - 定期更新模板图片以适应微信界面变化

## 📝 技术实现说明

本工具结合多种技术实现全自动化操作：

1. **PyAutoGUI**：模拟键盘鼠标输入
2. **OpenCV**：图像识别，用于定位界面元素
3. **多线程技术**：用于窗口焦点锁定，避免操作中断
4. **剪贴板技术**：确保中文和特殊字符的可靠输入
5. **PyGetWindow**：窗口管理和激活

这种多层次的技术融合，确保了工具在不同环境和条件下的稳定运行。通过可配置的搜索方式优先级和自定义快捷键，能够灵活应对不同界面布局和微信版本的变化。
